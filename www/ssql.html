Description

This extension provides a Scheme syntax for SQL. It should be considered experimental as the API is still incomplete and very much in flux. The code is located at https://the-brannons.com/fossil/ssql.

Authors

Original idea and implementation foundation by Peter Bex. Port to Chicken 4, addition of new features and maintenance by Moritz Heidkamp. Ported to CHICKEN 5, new features and maintenance by Chris Brannon.

Documentation

To be done, see the unit tests for now.

ssql->sql

[procedure] (ssql->sql connection ssql)

Transforms the SSQL code (represented as list) into the SQL query (represented as string). If connection is given, it will be used to escape strings in the query. #f can be passed to the connection parameter.

(ssql->sql #f '(select (columns id title author) (from books)))

SSQL syntax

Select queries:

(select (columns id title author) (from books) (where (= id 10)))

Insert queries:

(insert (into books)
        (columns title author)
        (values #("Novaja ziamla" "Janka Kupała")
                #("Vianok" "Maksim Bahdanovič")))
